CMSC5702 – Assignment 3: Cloud Computing

MPF Compare Hero

MPF Compare Hero is an application that pull data (through web scrapping techique) from other websites and convert them into structured data. The application has the following features.

List the MPF fund type

Show the MPF statistics based on the fund type

Recommend for me

A risk assessment test to select the funds that are suitable for you

Compare MPF Schemes

Allow users to choose 2 MPF funds, facilitate the comparsion by different indicators, e.g. Fund-to-Expense-Ratio , Annualized return , Risk Indicators

MPF Press

Show the latest Press from MPFA

The application is now hosted on Amazon AWS Cloud. The enviornment is setup with the deployment service tool AWS Elastic Beanstalk with Java supported.

Learn more on the architecture

AWS Architecture

Below shows the AWS Beanstalk environment

AWS Beanstalk environment consists of the web-tier and the data tier . Since it is a AWS free-tier, only 1 Amazon EC2 Instance (EC) is used and load-balancing / auto-scaling feature has not been configured.

More about the configurations

Configuration Details

Web Tier

  • Amazon Linux running Java 8
  • Environment type: Single instancer
  • Instance Type: t2.micro
  • Application server: tomcat

Data Tier

  • Amazon Relational Database Service (RDS) for the data persistent
  • Engine: mysql

Security

  • 1 Virtual Private Cloud
  • 1 SecurityGroup for ElasticBeanstalk environment
    • inbound rule: allows all with TCP port=80 from all
    • inbound rule: allows MySql TCP port = 3306 from RDS DB Security Group
  • 1 SecurityGroup for RDS DB Security Group
    • inbound rule: allows MySql TCP port = 3306 from the ElasticBeanstalk environment security group
